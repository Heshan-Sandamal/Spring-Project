# Spring MVC Maven Multi-Module Project

## Build the application


```
git clone https://github.com/niroshn/Spring-Project

mvn clean install
 
 ```
 



## Run the application in tomcat

Go to /web directory 

```
    cd /web

 ```
execute the command 


```
    mvn tomcat7:run

 ```
The application is deployed at localhost:8080
